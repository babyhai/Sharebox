# Sharebox

分享资料的站点；利用Ruby on Rails 写的
 用户可以上传分享自己的资料，也可以下载其他用户上传的资料.



* Ruby 2.3.0

* Ruby on Rails 5.1.14

* Postgresql 9.5

* 文件后期后保存在七牛云上目前是存在本地
